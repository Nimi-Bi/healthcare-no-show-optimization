# healthcare-no-show-optimization
Healthcare Business Intelligence project analyzing patient no-shows, appointment attendance, and revenue loss using Excel, SQL, and Power BI.

## Project Overview

This project analyzes patient appointment no-shows in a healthcare setting.

## Tools Used

- Excel
- MySQL
- Power BI
- GitHub

## Objectives

- Identify drivers of patient no-shows
- Measure financial impact
- Evaluate SMS reminder effectiveness
- Analyze departmental performance

## Key Findings

- Cardiology had the highest revenue loss
- Longer waiting times increased no-shows
- SMS reminders improved attendance rates
